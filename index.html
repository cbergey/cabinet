Biography

The language we use does not reflect the world; it selectively picks out remarkable facets of it. Given this, how do we use language to learn about the world?

My current projects examine how parents selectively remark on the world to young children, and how children learn from this selective description; how adults interpret description to make inferences about new categories; and how selective description affects computational models' representations of word meaning.
